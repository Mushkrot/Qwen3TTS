# Voice: Aiden

Built-in Qwen3TTS CustomVoice speaker used as an English long-form reading
baseline.

```text
Aiden/
  Input/
  Ready/
```

- `Input/`: reserved for source/reference audio if this built-in voice is ever
  used as a comparison reference.
- `Ready/`: local generated samples. The current useful listening pack was
  copied to `Ready/builtin_quality_2026-06-23/`.

The generated WAV files are intentionally ignored by Git.
